# GWgap_predictor_data

This page includes data that was used for constructing machine learning models in J. Lee et al., Phys. Rev. B 93, 115104 (2016) worked in Kyoto University.

Direct link : https://journals.aps.org/prb/abstract/10.1103/PhysRevB.93.115104
